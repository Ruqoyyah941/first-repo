Measurement Units In CSS

Absolute Lengths:
  1. pixels................px
  2. centimeters.......cm
  3. millimeters........m
  4. inches................in
  5. points.................pt
  6. picas..................pc
Relative Lengths:
  1. em........Relative to the font-size of the element (2em means 2 times the size of the current font
  2. ex..........Relative to the x-height of the current font
  3. ch..........Relative to the width of the "0" (zero)
  4. rem......Relative to font-size of the root element
  5. vw.......Relative to 1% of the width of the newport
  6. vh.......Relative to 1% of the height of the newport
  7. vmin.......Relative to 1% of viewport's smaller dimension
  8. vmax........Relaive to 1% of viewport's larger dimension
  9. %..........Relaive to the parent element

Differences between Absolute and Relative Lengths

Absolute length units are fixed and a length expressed in any of these will appear as exactly that size while Relative length units specify a length relative to another length property. Relative length units scale better between different rendering medium.

Difference between Px, em and %

Px is a static measurement, while % and em are relative measurements. % depends on its parent font size while em is relative to the current font size of the elements
