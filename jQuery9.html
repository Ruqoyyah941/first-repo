To unwrap means to remove the parent element and in this example, the parent element is the containerDiv as shown in the console So to unwrap, use $('div').unwrap (close the wrapped with a semi-colon and unwrap)

WrapAll wraps all the html elements with the specified element together as one while wrap wraps each element one by one

WrapInner wraps the html element with the intended element but puts it in the middle unlike wrap that makes it the parent element
